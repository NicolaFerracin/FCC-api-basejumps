API Basejump: Image Search Abstraction Layer

User stories:
  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example creation usage:

https://fast-hollows-1805.herokuapp.com/api/imagesearch/lolcats%20funny
https://fast-hollows-1805.herokuapp.com/api/imagesearch/lolcats%20funny?offset=3

Usage:

https://fast-hollows-1805.herokuapp.com/api/imagesearch/lolcats%20funny

Will return the 10 first results



https://fast-hollows-1805.herokuapp.com/api/imagesearch/lolcats%20funny?offset=3

Will return the first 10 results starting from the 2nd one. offset=1 would return the first 10 results starting from 0.



https://fast-hollows-1805.herokuapp.com/api/latest/imagesearch/

Will return the latest 10 searches made.
